jeforth.3ce background page We need a background page, because 3ce extension pages may not be existing at all while target page attached. In that case, readTextFile command from target pages need a service which must be always there and that's what background page is for. o jeforth.3ce.html is the index.html home page of popup page and extension pages, o jeforth.3ce.js is the Javascript portion of the jeforth.3ce.html home page. o jeforth.3ce.background.html is the index.html of the background page
